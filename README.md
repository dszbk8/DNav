# FCGame - FC/NES模拟器

## 按键说明
移动端使用触摸操控，PC端按键映射如下：
| 按键 | player1  |	player2 |
|:--------:| :---------:|:--------:|
| Left | A | Num-4 |
| Right | D | Num-6 |
| UP | W | Num-8 |
| Down | S | Num-2 |
| A | J |	Num-7 |
| B | K | Num-9 |
| AA | Z | Num-/ |
| BB | X | Num-* |
| Start | Enter | Num-1 |
| Select | Ctrl | Num-3 |
